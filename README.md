# Website Profil Dinamis

Ini adalah template website profil pribadi yang dirancang modern dan dinamis dengan animasi halus:

Fitur utama:
- Header navigasi dengan fade-in saat halaman dimuat.
- Foto profil dengan efek zoom-in dan rotasi lembut saat halaman dimuat.
- Setiap bagian (About, Skills, Projects, Contact) muncul dengan animasi fade-in-up saat di-scroll, menggunakan efek staggered.
- Kartu keahlian bereaksi saat di-hover dengan efek tilt; klik/fokus kartu akan membalik (flip) untuk menampilkan detail (akses keyboard friendly).
- Tombol CTA memiliki efek pulse subtil yang mengundang interaksi.
- Memperhatikan prefers-reduced-motion untuk pengguna yang mengurangi animasi.

Cara pakai:
1. Letakkan berkas `index.html`, `styles.css`, `script.js` dan folder `assets/` di satu folder.
2. Ganti `assets/profile.jpg` dengan foto profil Anda (atau ubah path di HTML).
3. Buka `index.html` di browser.

Penyesuaian:
- Ganti teks [Nama Anda], [profesi], dan deskripsi lain sesuai profil Anda.
- Tambah / edit kartu keahlian dan proyek menurut kebutuhan.
- Untuk deploy: Anda dapat meng-host statis di GitHub Pages, Netlify, Vercel, dsb.

Aksesibilitas:
- Kartus dapat diakses via keyboard (tab + Enter untuk membalik).
- Jika pengguna memilih prefers-reduced-motion, animasi dan transisi akan diminimalkan.

Jika ingin, saya bisa:
- Menambahkan form backend sederhana (contoh Netlify Forms atau endpoint server).
- Mengonversi ini menjadi template React/Vue jika Anda butuh interaktivitas lebih kompleks.
- Membuat varian warna / tema gelap-terang.
